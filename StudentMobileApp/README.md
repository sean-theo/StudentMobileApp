Student Mobile App

A .NET MAUI Android application developed as part of the WGU Software Engineering Capstone (D424).
This app allows students to track academic terms, courses, and assessments with local data storage and basic notifications.

Features

Add, edit, and delete terms, courses, and assessments

Input validation for all form fields

Local SQLite database storage

Search and reporting functionality

Notifications for important course or assessment dates

Simple, user-friendly interface

Installation (Local Development)

Clone this repository:
git clone https://gitlab.com/wgu-gitlab-environment/student-repos/stheob9/d424-software-engineering-capstone.git

Open the solution in Visual Studio 2022 (Community or higher).

Ensure the following workloads are installed:

.NET Multi-platform App UI development (MAUI)

Android SDK tools

Build the project and run on an Android emulator or connected device.

Installation (Android APK)
Once the APK is hosted, download the file and install it on your Android device.
Make sure “Install unknown apps” is enabled in system settings.
